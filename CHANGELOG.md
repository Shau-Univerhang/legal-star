# 更新日志

## [0.2.2] - 2026-03-02

### 🔄 架构与部署优化
- **静态资源迁移**：将前端静态资源从 `appui/appui` 彻底迁移至 `public/appui`，作为单一事实来源（Source of Truth），解决了 Vercel 部署时更新不生效的问题。
- **目录清理**：删除了冗余的 `appui/appui` 开发目录，降低维护成本，避免代码版本冲突。
- **文档更新**：更新 `README.md`，明确了新的目录结构和开发/部署流程。

### 🐛 问题修复
- **验证码登录修复**：增强了后端验证码校验逻辑（`api/auth/verify-code`），现在能自动处理 `email`（登录）、`signup`（新用户注册）和 `recovery`（找回密码）等多种 OTP 类型，解决了新用户无法通过验证码登录的问题。
- **导航栏统一**：修复了所有子页面（如案例库、题库、个人中心等）的左侧导航栏，确保“AI咨询”入口在所有页面均可见且高亮状态正确。

## [0.2.1] - 2026-02-24

### 🔄 合并与集成
- **合并小伙伴代码**：成功集成了另一版本的项目代码，保留了双方的核心功能。
- **AI 智能咨询**：集成了 Coze AI 聊天功能，新增了 `P-AI_CHAT.html` 页面和相关 API 支持。
- **自动跳转**：访问根路径 `/` 时现在会自动重定向到 `/appui/P-HOME.html`，提升用户体验。

### 🚀 新增功能
- **邮箱验证码注册**：实现了完整的邮箱验证码（OTP）注册流程，取代了原有的链接确认方式，用户体验更流畅。
- **自定义 SMTP 支持**：后端适配了 Supabase 的自定义 SMTP 设置（推荐使用 Resend），解决了默认邮件服务的发送速率限制问题。
- **静态资源自动部署**：优化了前端静态资源（`appui`）的部署策略，现在构建时会自动同步到 `public/` 目录，确保 Vercel 等平台部署无 404 问题。

### 💅 UI/UX 优化
- **首页导航升级**：在侧边栏新增了“AI咨询”入口。
- **路径修复**：修复了首页中存在的本地绝对路径（`file:///`），确保在所有环境中链接正常跳转。
- **验证码输入优化**：更新了注册页面的验证码输入框，现在支持 6-10 位验证码（完美适配 Supabase/Resend 发送的 8 位验证码）。
- **界面精简**：移除了未使用的手机号注册选项，专注于提供稳定的邮箱注册服务。

### 🔧 工程化与修复
- **依赖升级**：升级 `next`、`eslint` 等核心依赖至最新版本，修复了构建警告和潜在安全问题。
- **构建修复**：解决了 `eslint-config-next` 版本冲突导致的 Vercel 构建失败问题。
- **部署配置**：更新了 `package.json` 和构建脚本，确保生产环境静态资源正确加载。
